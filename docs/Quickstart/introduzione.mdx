# penguins-eggs il "Rinascimento" nella rimasterizzazione?

In Italia abbiamo a cuore il **Rinascimento**, periodo in cui fiorirono arti e cultura. 
Nel mondo Linux, abbiamo vissuto un'epoca d'oro della rimasterizzazione con strumenti 
come *Remastersys* e *Systemback*. 
Oggi, **penguins-eggs** si propone come l'erede naturale di quel periodo: 
un "Rinascimento" moderno e universale dei sistemi di rimasterizzazione.


---

## 🐧 Che cos'è penguins-eggs?

Sviluppato da Piero Proietti, **penguins-eggs** è uno strumento di clonazione e 
rimasterizzazione universale. Il nome evoca la "riproduzione": come un pinguino 
che cova le sue uova, il tool permette di "far schiudere" nuovi sistemi partendo 
da quelli esistenti.

### Perché è unico?

* **Universale**: Supporta Debian, Ubuntu, Arch Linux, Fedora, Manjaro e openSUSE.
* **Multitasking**: Gestisce sia sistemi desktop grafici che server e minimali (CLI).
* **Sicuro**: Utilizza esclusivamente pacchetti originali delle distribuzioni, garantendo integrità e affidabilità.

---

## 🚀 Funzionalità in breve

| Caratteristica | Comando | Descrizione |
| :--- | :--- | :--- |
| **Standard** | `produce` | Crea una ISO pulita senza dati personali. |
| **Clone** | `--clone` | Preserva dati e configurazioni (ottimo per backup). |
| **Crypted** | `--homecrypt` / `--fullcrypt` | Salva i dati personali in immagini LUKS crittografate. |
| **Wardrobe** | `wardrobe` | Gestisce "costumi" per automatizzare la creazione. |
| **Cuckoo** | `cuckoo` | Server PXE integrato per installazioni via rete. |
