## ü¶ë Calamares: L'installer grafico (GUI)

Se hai creato una ISO con un'interfaccia grafica (XFCE, GNOME, KDE, ecc.), puoi offrire un'esperienza di installazione professionale grazie a **Calamares**.

* **Quando usarlo**: Per distribuzioni destinate all'utente finale o se desideri un partizionamento grafico assistito.
* **Come attivarlo**: Eggs integra Calamares automaticamente se rileva un ambiente grafico. Puoi configurarlo tramite i **Themes** del Wardrobe.
* **Comando**:
    ```bash
    sudo eggs calamares
    ```

## ü§ñ Installazione Unattended (Automatica)

Per gli utenti avanzati che devono installare molti computer identici, eggs permette l'installazione **Unattended**.

Utilizzando i file di configurazione YAML presenti nel Wardrobe (cartella `config`), puoi avviare un'installazione che non richiede alcun intervento umano:

```bash
sudo eggs install --unattended
```

:::tip Personalizzazione
Puoi creare il tuo file di configurazione (es. `it.yaml`) per impostare automaticamente la lingua italiana, il fuso orario e le credenziali utente durante l'installazione.
:::
