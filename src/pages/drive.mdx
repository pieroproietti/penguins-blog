## Packages
The best way to install `penguins-eggs` on your distro is probably to use `get-eggs`:
* install git if not already available
* `git clone https://github.com/pieroproietti/get-eggs`
* `cd get-eggs`
* `sudo ./get-eggs`
* and follow instructions.

If you prefere, You can also download the packages from 
[penguins-eggs.net/basket](https://penguins-eggs.net/basket/index.php/packages/?p=packages) and install them.
